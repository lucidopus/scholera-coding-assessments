# Scholera — Coding Assessments

This repository contains take-home assessments for engineering candidates at Scholera.

If you have been sent here, head to the `assessments/` folder and open the file that matches your role:

- **AI Engineer** → [`assessments/ai-engineer.md`](assessments/ai-engineer.md)
- **Mobile Developer** → [`assessments/mobile-developer.md`](assessments/mobile-developer.md)

> This repository is read-only. Do not push your work here — create a brand new public repository of your own.
